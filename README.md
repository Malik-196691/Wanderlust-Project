# Wanderlust

**Wanderlust** is a web application for travelers to explore destinations, plan trips, and share experiences.


---

## Features

- User authentication and profiles  
- Explore and search destinations  
- Create and save travel itineraries  
- Bookmark favorite places  
- Comment and review destinations  

---

## Tech Stack

- **Frontend:** Bootstrap  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Other Tools:** Cloudinary, JWT
